Help section

0) Machine configuration labels must be one character only

1) The only empty section can be the SCANNED TAPE SYMBOL, it implies a blank tape but will appear as a hash in the table

2) Please be explicit about doing nothing, that means use RL or LR in the actions section if you want to do nothing

3) R is the action to go right, L is the action to go left, P will print anything after it and E will erase anything
