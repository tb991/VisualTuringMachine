Help section

0) Machine configuration labels must be one character only

1) The only empty sections can be the SCANNED TAPE SYMBOL and the ACTIONS

2) R is the action to go right, L is the action to go left, P will print anything directly after it and E will erase anything

3) An asterisk can be used to imply any symbol (except blank/#) in the READ section

4) A ! can be used to imply not(symbol) in the READ section

Inspired by the earlier chapters of the book The Annotated Turing by Charles Petzold
